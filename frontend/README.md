# DSA Playground (Frontend)

Interactive visualizations for sorting algorithms and basic data structures.

## Quick start

1. Open a terminal and change to the frontend folder:

```powershell
cd "c:\Users\HP\Music\DSA Mini Project\frontend"
```

2. Install dependencies and run the dev server:

```powershell
npm install
npm run dev
```

3. Open the printed local URL (usually http://localhost:5173) in your browser.

## Features included (initial)

- Sorting visualizer (Bubble Sort implemented step-by-step).
- Linked List visualizer (insert at start/end, delete by value, reverse).
- Stack simulator (push/pop/peek) with overflow/underflow messages.
- Queue simulator (enqueue/dequeue/peek) with front/rear visualization.
- Light/Dark mode toggle.
- Framer Motion animations and Tailwind styling.
